# meteor-pickadate
pickadate package for Meteor 

http://amsul.ca/pickadate.js/

Current version `3.5.6`

## Usage

- insert a input
```
<input type="text" id="input_id">
```

- to create a date picker
```
$('#input_id').pickadate(opts)
```

- to create a time picker
```
$('#input_id').pickatime(opts)
